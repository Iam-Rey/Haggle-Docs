Authentication

Haggle allows both the authenticated and unauthenticated user access to the platform with the authenticated user having full access to the platform. A user is required to be authenticated before granted full access to the platform.

Unauthenticated user (User flow)

When an unauthenticated user visits the landing page, the user would be able to view and interact basically with the page. The user would be able to browse through available products by using the search filter function or by using the category section for easy navigation through products. When the unauthenticated user selects a product, the user will then be directed to the product page that’ll contain the specific product from different merchants with their prices. The unauthenticated user will only be able to view and compare prices from 2 different merchants. On request to “view more prices", the unauthenticated user will get redirected to the login page.

Authenticated User (User flow)

After a user becomes authenticated, the user gets full access to the platform. The user will be able to browse through available products and get price comparisons from all available platforms. After authentication, several options like comment boxes and the share options that are not available to the unauthenticated user will become available. The authenticated user will be able to comment directly under products and prices. Also available on the product page of the authenticated user will be the link to the purchase page of the product from the selected platform. The user will be redirected to the e-commerce platform for product purchase once the link is clicked.

Authentication Guide

Login

  1. Click on Sign In/Register on the top right corner of the page, It'll take you to the login page.
  2. In the login page, you'll be required to input your username and your password.
  3. After inputting your details, click the login button below, and you'll be logged in.
  4. You can then interact with the page freely as you'd have been authenticated.

Sign Up

  1. Click on Sign In/Register on the top right corner of the page, It'll take you to the login page.
  2. Click on the sign up option beside the login option and you'll be taken to the sign up page.
  3. At the sign up page, you'll be required to input your First Name, Last Name, your email, username and create a password which will be required of you upon every new attempt to login later.
  4. After inputting your details, click the sign up button below and you'll be registered as a user in Haggle database.
  5. On successful registeration, you'll then be redirected to the login page, where you'll be required to login with your username and password.
